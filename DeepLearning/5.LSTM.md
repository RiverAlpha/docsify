# LSTM

## 架构

LSTM 通过 4 个核心组件来更新这两个状态：

遗忘门 $F_t$（决定从长期记忆 $C_{t-1}$ 中丢弃什么）：$$F_t = \sigma(X_t W_{xf} + H_{t-1} W_{hf} + b_f)$$

输入门 $I_t$（决定让哪些新的信息进入长期记忆）：$$I_t = \sigma(X_t W_{xi} + H_{t-1} W_{hi} + b_i)$$

候选记忆 $ \tilde{C}_t$（当前时间步提取出的新知识）：

$$ \tilde{C}_t = \tanh(X_tW_{xc} + H_\{t-1\}W_{hc}+b_c) $$

输出门 $O_t$（决定把长期记忆的哪些部分提取出来作为短期记忆 $H_t$）：$$O_t = \sigma(X_t W_{xo} + H_{t-1} W_{ho} + b_o)$$

最终状态更新公式：

更新长期记忆：$C_t = F_t \odot C_{t-1} + I_t \odot \tilde{C}_t$

更新短期记忆：$H_t = O_t \odot \tanh(C_t)$

(注：$\odot$ 表示张量的逐元素相乘，在代码中直接用 * 即可。)

## 代码

```python
import torch
import torch.nn as nn

class MyLSTM(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super().__init__() # 使用现代 Python 3 语法
        self.hidden_size = hidden_size
        
        # --- 核心技巧：合并权重 ---
        # 我们把 Input, Forget, Cell, Output 4个门的权重拼成一个大矩阵
        # 矩阵维度是: (输入维度, 隐藏层维度 * 4)
        self.W_x = nn.Parameter(torch.randn(input_size, hidden_size * 4) * 0.01)
        self.W_h = nn.Parameter(torch.randn(hidden_size, hidden_size * 4) * 0.01)
        
        # 偏置项也拼起来，长度为 hidden_size * 4
        self.b = nn.Parameter(torch.zeros(hidden_size * 4))
        
        # 输出层的权重和偏置 (和之前的 RNN 一样)
        self.W_hq = nn.Parameter(torch.randn(hidden_size, output_size) * 0.01)
        self.b_q = nn.Parameter(torch.zeros(output_size))

    def forward(self, X, state=None):
        batch_size, seq_len, _ = X.size()
        
        # LSTM 的 state 是一个元组，包含 (H, C)
        if state is None:
            H = torch.zeros(batch_size, self.hidden_size, device=X.device)
            C = torch.zeros(batch_size, self.hidden_size, device=X.device)
        else:
            H, C = state
            
        outputs = []
        
        for t in range(seq_len):
            X_t = X[:, t, :]
            
            # 1. 一次性进行大矩阵乘法，算出 4 个门的所有线性结果
            # gates 的形状: (batch_size, hidden_size * 4)
            gates = torch.matmul(X_t, self.W_x) + torch.matmul(H, self.W_h) + self.b
            
            # 2. 沿着维度 1 (dim=1) 把结果均分为 4 块，每块形状是 (batch_size, hidden_size)
            # 这就完美对应了公式里的 i, f, g(候选C), o
            I_gate, F_gate, C_tilde, O_gate = gates.chunk(4, dim=1)
            
            # 3. 施加激活函数 (对应公式 1, 2, 3, 4)
            I = torch.sigmoid(I_gate)        # 输入门 (Sigmoid 输出 0~1)
            F = torch.sigmoid(F_gate)        # 遗忘门 (Sigmoid 输出 0~1)
            C_tilde = torch.tanh(C_tilde)    # 候选记忆 (Tanh 输出 -1~1)
            O = torch.sigmoid(O_gate)        # 输出门 (Sigmoid 输出 0~1)
            
            # 4. 更新细胞状态 C (核心传送带)
            # F * C: 遗忘过去的某些记忆; I * C_tilde: 写入当前的某些新记忆
            C = F * C + I * C_tilde
            
            # 5. 更新隐藏状态 H
            H = O * torch.tanh(C)
            
            # 6. 计算当前步的输出 O_t
            out_t = torch.matmul(H, self.W_hq) + self.b_q
            outputs.append(out_t)
            
        # 注意：返回的 state 变成了 (H, C) 元组
        return torch.stack(outputs, dim=1), (H, C)

```
